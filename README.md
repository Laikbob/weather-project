</p>

<h2>Rakenduse lehed</h2>

<h3>
  Uudiste leht – kuvab viimased uudised mugavas voos, mida on lihtne sirvida ja lugeda.<br>
  Ilmaleht – näitab ilmastikuandmeid graafikute ja kaardi abil, sisaldab prognoosi järgmiste päevade jaoks.
</h3>

<h2>Tehnoloogiad</h2>

<h3>
  PHP – serveripoolse loogika jaoks<br>
  JavaScript – interaktiivsete elementide ja graafikute loomiseks<br>
  SQL – andmete salvestamiseks (uudised ja ilmainfo)<br>
  HTML/CSS – veebilehe struktuuri ja kujunduse loomiseks
</h3>
